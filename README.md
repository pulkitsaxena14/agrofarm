AgroFarm - Community driven communication system.

Tools and Technologies Used:
Flask, PostgreSQL, SQLAlchemy, Elasticsearch, Bootstrap, JavaScript, jQuery, AJAX

A web based application which provides a platform for the members of the agricultural community worldwide to ask, answer and vote content. It also provides auto-recommendations to users for any unanswered questions.
